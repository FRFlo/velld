# Velld

**Velld** is a self-hosted database backup management and automation tool. It provides a modern, user-friendly web interface for scheduling, managing, and monitoring backups across multiple databases — helping you ensure data safety, automate routine backup jobs, and simplify disaster recovery.

## Why Velld?

- ✅ **Self-hosted**: Keep full control of your data and infrastructure.
- 📆 **Scheduled Backups**: Automate backups with custom schedules.
- 🔐 **Secure by Design**: Built-in encryption and access controls.
- 📊 **Real-time Monitoring**: Get a clear view of backup status and history.
- 💬 **Multi-database Support**: Works with PostgreSQL, MySQL, MongoDB, and more.

## Supported Databases

- PostgreSQL
- MySQL
- MongoDB

## Quickstart

Velld is Docker-first for ease of deployment:

```bash
git clone https://github.com/dendianugerah/velld.git
cd velld
docker compose up -d
```
